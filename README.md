# Template Repository for React app

> ### Main technologies
>> TypeScript
>> 
>> SASS / SCSS
>> 
>> Redux

> ### Configs
>> ESLint
>> 
>> Prettier
>> 
>> Husky(pre-commit git hook)

## Get Yarn
```
$ npm install --global yarn
```
## Getting started with Yarn
>> `yarn` - install dependencies 
>> 
>> `yarn start` - start app 

## Useful scripts
> Run ESLint
```
yarn lint | yarn lint:fix - run ESLint
```
> Run Pretiter 
```
yarn format
``` 
> Run ESLint & Prettier
```
yarn push-prepare
```
